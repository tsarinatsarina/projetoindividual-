No HTML, temos uma estrutura básica com as tags <!DOCTYPE html>, <html>, <head> e <body>. O <head> contém metadados, como o título da página, a codificação de caracteres e os links para a folha de estilos CSS e as fontes. No <body>, temos o cabeçalho do site, com um título <h1> e uma barra de navegação <nav> contendo links para as páginas Home, Sobre e Contato. Em seguida, temos uma imagem que serve como divisor de página, seguida pela seção principal da página, onde há uma imagem minha e um parágrafo com informações sobre mim. Após outra imagem de divisor de página, temos o rodapé da página, contendo um parágrafo com o texto "Um site por Malena Stefano".

No CSS há definições para o cabeçalho <h1>, a barra de navegação utilizando-se do hover e criando uma sombra  <nav>, as listas <ul> e <li>, as imagens com a classe divider e imgMalena, a seção principal <main>, e o rodapé <footer>.
